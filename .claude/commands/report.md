Erstelle mir eine Gesamtzusammenfassung der Änderungen auf englisch, damit ich diese in einem MergeRequest als Description verwenden kann. Führe nur die wesentlichen Punkte dabei auf.

Bitte strukturiere die Description wie folgt:
- **Summary**: Kurze Zusammenfassung (1-2 Sätze)
- **Changes**: Auflistung der wichtigsten Änderungen
- **Technical Details**: Relevante technische Details falls nötig
- **Testing**: Wie wurde getestet / wie kann getestet werden

Halte dich kurz und prägnant - fokussiere auf das Wesentliche für Code-Reviewer.
